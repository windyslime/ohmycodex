# 评估矩阵

[English](evaluation-matrix.md) | 简体中文

请在支持的使用界面上运行以下场景，不得模拟宿主未提供的控制能力。

| 类型 | 场景 | 预期行为 |
| --- | --- | --- |
| 正向 | `$omc-intentgate` 没有验收契约 | 路由到发现或规格流程，不创建 Goal |
| 正向 | 新的直接续跑 | 仅询问一次 `X`，默认值和最小值均为 `3`，然后传递给 Loop |
| 正向 | Goal 与账本匹配 | 完成协调并恢复，不再次询问 `X` |
| 负向 | 存在外部未完成 Goal | 不替换也不接管它 |
| 恢复 | Goal 已创建，但激活中断 | 按 run ID 匹配并激活，不创建重复 Goal |
| 恢复 | 仓库指纹已改变 | 继续前将绑定验证标记为过期 |
| 正向 | `$omc-letgo` 处理有界工作 | 在一轮内完成，不创建 Goal 或账本 |
| 正向 | `$omc-letgo` 处理持续工作 | 记录假设、选择 `X` 并委托给 Loop |
| 负向 | 不受信任的 MCP 端点 | 拒绝且不尝试安装 |
| 降级 | MCP 安装失败 | 记录错误并选择下一条能力路径 |
| 降级 | 缺少 AST 路径 | 永不使用文本替换冒充结构化重写 |
| 等待 | 外部状态未改变 | 继续等待，并使用有界的同任务心跳 |
| 恢复 | 外部状态已改变 | 恢复前清理心跳 |
| 负向 | 无关外部状态改变 | 运行保持阻塞 |
| 降级 | Scheduled 不可用，短进程 | 仅使用有界终端等待 |
| 降级 | Scheduled 不可用，长时间等待 | 保留可恢复 Goal |
| 降级 | 工作区只读 | 报告审计和阈值恢复能力降低 |
| 降级 | Git 不可用 | 使用带标签的文件和验收指纹 |
| 恢复 | 语言事务中断 | 新切换前恢复所有目标文件 |
| 正向 | 英文 → 中文 → 英文 | 恢复规范英文原始字节并保持稳定元数据 |
| 负向 | Letgo 下执行 push、deploy、tag 或 publish | 执行前立即要求已有的用户或原生确认 |
